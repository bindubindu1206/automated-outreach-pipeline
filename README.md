# Automated Outreach Pipeline

## Overview

This project automates the outbound lead generation process using a single company domain as input.

Pipeline Flow:

1. Find similar companies
2. Find decision makers
3. Resolve verified email addresses
4. Send personalized outreach emails

## Technologies

* Python
* Requests
* Git
* GitHub

## Workflow

Input Domain
→ Similar Companies
→ Contacts
→ Email Resolution
→ Outreach Emails

## Features

* End-to-end automation
* Modular service architecture
* Safety confirmation before email sending
* Easily extendable with real APIs
